Installation Guide
1
Go to GHL Agency Settings → Company → Custom JS and paste this — adds CliqLabs to your sidebar:
<script src="https://cdn.thecliqlabs.com/loader.js"></script>
2
Go to GHL Agency Settings → Company → Custom CSS and paste the CSS code from Step 3 above.
3
Click Save Changes then hard refresh GHL (Ctrl+Shift+R or Cmd+Shift+R).
4
✅ Your GHL is now branded! Come back here anytime to change your theme.
